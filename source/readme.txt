== Description ==

Generates links for Analytics tools and short link. Enter your Campaign Name, Source, Medium (UTM link) etc. to generate a full link and a short link (trough the Google URL Shortener API) all in once. <https://ga-dev-tools.appspot.com/campaign-url-builder/>

Available languages :
* English
